## 🧩 What this app does

cheatsheet_machines_htb is an interactive dashboard for Hack The Box machines. It shows hacked machines, the techniques used, and the difficulty level in one place.

It is built for quick review. You can use it to browse data, compare machines, and keep track of your progress with clear visual badges and a dark theme.

## 🚀 What you need

Use this app on Windows with a modern web browser.

You will need:

- Windows 10 or Windows 11
- Google Chrome, Microsoft Edge, or Firefox
- An internet connection for the first download
- A few megabytes of free disk space

If you plan to keep your machine list updated, Excel can help with data review before you import or edit entries.

1. Open the download link above in your browser.
2. On the GitHub page, look for the green **Code** button.
3. Click **Code** and choose **Download ZIP**.
4. Save the ZIP file to your PC.
5. Right-click the ZIP file and choose **Extract All**.
6. Open the extracted folder.
7. Look for the main HTML file or project start file.
8. Double-click the file to open it in your browser.

If the app is hosted as a web page, you can also open it straight from the link and use it without installing extra software.

## 🏁 How to use it

1. Open the dashboard in your browser.
2. Browse the machine table.
3. Use search to find a machine name.
4. Sort by difficulty to group easy, medium, and hard entries.
5. Review the techniques used for each machine.
6. Check the badges for fast visual reference.
7. Open each entry to inspect the machine details.

This layout helps you scan many machines fast. You can use it during study sessions, note taking, or CTF review.

## 🔍 Troubleshooting

If the page does not open:

- Check that you extracted the ZIP file first
- Make sure you opened the correct file in the folder
- Try a different browser
- Refresh the page
- Confirm that the files stay together in the same folder

If the table looks empty:

- Check the data file
- Make sure the file name did not change
- Open the browser console only if you know how to use it
- Re-download the project and try again

If badges do not show:

- Reload the page
- Check your internet connection
- Open the app in Chrome or Edge
